# 1 概述
## 1.1 分布式存储概念
特性：
* 可扩展
* 低成本
* 高性能
* 易用

技术要求：

*  数据分布
*  一致性
*  容错
*  负载均衡
*  事务与并发控制
*  易用性
*  压缩/解压缩

## 1.2 分布式存储分类
数据需求分为三类：
* 非结构化数据
* 结构化数据
* 半结构化数据

分布式存储系统分类：
* 分布式文件系统
* 分布式键值系统
* 分布式表格系统
* 分布式数据库

名词解释：
blog = binary large object 如图片、照片、视频等非结构化数据对象。
### 1.2.1 分布式文件系统
分布式文件系统存储三种类型的数据：Blob对象、定长块、大文件。
典型系统：Facebook haystack 、google file system 、amazon elastic block store
